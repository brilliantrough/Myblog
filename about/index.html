博客介绍#

本博客由 @pezayo 在2022年3月28日搭建。使用的网上非常流行 hexo 框架，使用的主题是 next 。

本博客之所以搭建，是因为作者本人平日总会突发奇想的去整活，风风火火的整了一通后，总是会忘了记录，类似于代码不写注释一样。于是下次再来的时候又得从头开始。所以作者本人开始养成了遇事记录的“好习惯”。而作者采用的记录方式便是 markdown，有关于各种软件层面的记录比如 python，pycharm，vscode，Altium Designer等。

后来作者还觉得不过瘾，以及作者在平时与他人接触的过程中，慢慢发现了身边许多人都拥有自己的博客。于是作者想着什么时候也能有自己的博客。心动不如行动，但此“大项目”一直因为无聊又恼人的作业，任务，以及其他项目被搁置。

终于作者在这么一个风雨交加的夜晚，在室友熟睡（刷视频？）之际晚上开始了自己的博客搭建，照着知乎上的简易教程，开始了自己的博客之旅。

域名#

此域名 pezayo.com 由作者在阿里以一年60人民币的价格购买。此博客托管在 github 上，将 github 上的工程重定向到这个网址。

头像#

头像在多方打听询问后，作者决定还是采用自己的老式头像。最初作者采用的是 Microsoft Visual Studio 提供的一个少女头像，但是由于比例问题，这个不是一个正方形，导致呈现在博客上头像被纵向拉长。觉得膈应，又换成了自己原先的头像。

友链#

暂时还没有向身边大佬申请建立友情链接。暂时将友链选定为作者常用且吹爆的 Stack Overflow，以及另一个某链接。

RSS 订阅#

限于作者水平问题，还没有开启 RSS 订阅服务。博客中显示的一个RSS的选项按钮，实则指向一个空地址。等作者日后水平有机会得以提升后，再与之一战。

看过其他大佬的 RSS 订阅功能，看着就感觉复杂，外加作者本身又非常忙，也抽不出空来，就搁置了。

背景选取#

背景为作者去了专门提供图片的网站， colorhub.me 去寻找的一个外太空卫星的图片。因为作者对这种外太空卫星很着迷，觉得很有一种科幻感。浪漫~~

但是也没有找到其他适合的背景了。

分类#

有关于博客文章的分类，作者并没有做的很系统。以及到底在上传哪些文本到博客上，作者也没去认真规划过。

就目前而言，作者一直在博客上写一些技术方面的文章，主要偏向软件方向。至于后来是否会上传一些心得感悟，社会，生活方面的文章还有待斟酌。或许这可以另开一个公众号来进行记录。

标签#

有关标签的选取更是随意。没有固定的标签。在写文本的时候想到什么就贴什么标签。至于贴了多少个标签，作者也不知道了。甚至会有把标签与分类混淆的时候。
